# TapGuard - Repeated Tap Blocker

OnePlus Android এর জন্য Repeated Tap Blocker অ্যাপ।

---

## APK বানানোর নিয়ম (GitHub দিয়ে, ফ্রি)

### ধাপ ১ - GitHub অ্যাকাউন্ট বানান
- https://github.com এ যান
- "Sign up" করুন (ফ্রি)

### ধাপ ২ - নতুন Repository বানান
- লগইন করার পর উপরে ডানে "+" বাটনে ক্লিক করুন
- "New repository" সিলেক্ট করুন
- Repository name দিন: `TapGuard`
- "Public" সিলেক্ট করুন
- "Create repository" ক্লিক করুন

### ধাপ ৩ - ফাইল আপলোড করুন
- Repository পেজে "uploading an existing file" লিংকে ক্লিক করুন
- এই ZIP এর সব ফাইল Extract করুন
- Extract করা **TapGuard ফোল্ডারের ভেতরের সব ফাইল** drag করে আপলোড করুন
  (ফোল্ডারটা নিজে না, ভেতরের সব ফাইল ও ফোল্ডার)
- নিচে "Commit changes" ক্লিক করুন

### ধাপ ৪ - Build শুরু হবে অটোমেটিক
- Repository পেজে "Actions" ট্যাবে ক্লিক করুন
- "Build TapGuard APK" দেখবেন - চলছে
- ৩-৫ মিনিট অপেক্ষা করুন
- সবুজ টিক আসলে সফল হয়েছে

### ধাপ ৫ - APK ডাউনলোড করুন
- Actions এ workflow-তে ক্লিক করুন
- নিচে "Artifacts" সেকশনে "TapGuard-Debug-APK" দেখবেন
- ক্লিক করে ডাউনলোড করুন!

---

## OnePlus-এ ইনস্টল করুন
1. Settings → Additional Settings → Developer Options → চালু করুন
2. Install Unknown Apps → চালু করুন
3. APK ফাইলে ট্যাপ করুন → Install
